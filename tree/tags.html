Read, Write, Include & Link

The tag system allows for quick operation and takes off a lot of work. Are the basic tags not own enough you can add your own tags.

Variables

Create persistent variables to pass informatiom from your page to your template

{hanya:write(key|value)} create a variable
{hanya:read(key)} read the Variable
{hanya:if(key|is|then)} checks the variable and prints on true

Database

Data in the database can be edited by the customer through the admin interface.

{hanya:db(string|id)} save a string to use with inline html elements
{hanya:db(text|id)} save a text to use with block html elements
{hanya:db(html|id)} save HTML sourcecode

Dynamic

Attach data to a block to render them later in the template with the block tag

{hanya:attach(id|file|name)} searches for a file like "elements/+name.html"
{hanya:attach(id|html|sourcecode)} appends the sourcecode to the block
{hanya:block(id)} renders the given block

Include a partial to outsource Navigation and Footers

{hanya:include(name)} searches for a file like "elements/_name.html"

URL

Simplify your handling with URL's

{hanya:file(name.ext)} gets URL for a file in the "files" directory
{hanya:link(dir/page)} gets URL for a page

Other

Some other tags

{hanya:template(id)} defines the template (should be on top of a page)
{hanya:benchmark()} renders the benchmark informations
{hanya:toolbar()} renders the admin toolbar
