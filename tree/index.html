Welcome to Hanya

Hanya is a powerful and simple Website Engine. As a developer, you have total control over the structure and appearance of the page, while customers can easily process small parts of the site through an admin interface. The simple expansion System lets you quickly add more features to the System. Hanya is a rapid and lighweight System to build fast Websites.

Rapid Webpage building

Hanya is a Website Engine no Content Management System, so you have to define the Structure of your Site with files. Every accesible Page is defined in a File in the "tree" directory. Hanya automatically maps the Request to a Page for you: 

http://www.example.com/ => "tree/index.html"
http://www.example.com/page/ => "tree/page.html"
http://www.example.com/directory/subpage => "tree/directory/subpage.html"

Tag-based Language

Hanya provides an understandable Tag system to keep your sourcefiles clean from Code. These Tags are available in every sourcefile and lets you focus on Design and Content. A Example Page sourcefile can look like this:

{hanya:write(title|Home)}
<h1>Example Page</h1>
<p>{hanya:db(text|welcome-text)}</p>
<div class="quicklinks">
  {hanya:include(quicklinks)}
</div>
{hanya:attach-to(sidebar|file|home)}

Hanya has a global Variables system that allow write and read actions from every sourcefile. This Mechanism allows you to pass data from a page to a template or block element. The "db" Tags is used for interaction with a database. All db tags can interactively manipualted in the Administrator Interface. The system has also Tags to include files and attach content to blocks which can be rendered later in a template.

Configuration

Set the Configuration in the index.php file in you root directory. The Database Location is a PHP PDO Location.

Hanya::run(array(
  "system.locale" => "de_CH",
  "system.timezone" => "Europe/Berlin",
  "db.location" => "mysql:dbname=hanya;unix_socket=/tmp/mysql.sock",
  "db.user" => "username",
  "db.pass" => "password",
  "admin.users" => array("admin"=>"admin"),
));
