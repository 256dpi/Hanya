Sitemap Generation

The automatic generation of the "sitemap.xml" is builtin. The generator checks the structure in the "tree" directory to buid the xml file.


  
    http://www.example.com/
  
  
    http://www.example.com/page
  
  
    http://www.example.com/directory/page

.htaccess

The .htaccess file comes with some important settings. The "Expire" settings are all set by default and the server trys to gzip compress all files. "ETags" are off so the Browser loads the tiny Site at every Request.
